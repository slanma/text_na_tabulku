Tady je verze 2! Co je nového oproti originálu:
Exporty — CSV, Excel (.xlsx), PDF, Markdown (.md), JSON — všechno jedním kliknutím.
Editace tabulky — dvojklik na buňku ji otevře k editaci, tlačítko "Režim úprav" zapne checkboxy pro výběr řádků, editaci záhlaví a mazání.
Správa struktury — přidání řádku, přidání sloupce (s vlastním názvem), smazání vybraných řádků.
Drag & drop — přetažení souboru (TXT, CSV, TSV, JSON) rovnou načte a převede.
